# Logo Image for RIZA

Please place your RIZA logo image file in this directory.

**Expected file name:** `riza-logo.png`

**Full path:** `d:\RIZA\client\src\assets\images\riza-logo.png`

**Supported formats:** PNG, JPG, SVG (PNG recommended for best quality with transparent background)

**Recommended size:** 
- For header: 200-300px width (height will scale automatically)
- For login page: 200-300px width

**Instructions:**
1. Save your logo image file
2. Rename it to exactly: `riza-logo.png`
3. Copy it to: `d:\RIZA\client\src\assets\images\riza-logo.png`
4. Run: `cd d:\RIZA\client && ng build`
5. Restart server: `cd d:\RIZA\server && node rizaserver.js`

If the logo file is not found, the page will automatically fall back to displaying the text "RIZA" with the tagline.
