# Icons Directory

This directory contains all icon images used in the RIZA restaurant application.

## Icon Naming Convention

Icons are named following this pattern:
- Category icons: `category-{id}.png` (e.g., `category-starters.png`)
- Menu item icons: `item-{name}.png` (e.g., `item-crispy-corn-chaat.png`)
- Feature icons: `feature-{name}.png` (e.g., `feature-fine-dining.png`)
- Side menu icons: `side-{id}.png` (e.g., `side-overview.png`)
- Social media icons: `social-{name}.png` (e.g., `social-instagram.png`)
