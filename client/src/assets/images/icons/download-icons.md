# Icon Download Instructions

To download icons for the RIZA restaurant application, you can use the following approach:

## Option 1: Manual Download
1. Visit https://unsplash.com or https://pexels.com
2. Search for food/restaurant related images
3. Download images and rename them according to the naming convention in README.md
4. Place all images in this directory (assets/images/icons/)

## Option 2: Using a Script
You can create a simple script to download icons from Unsplash using their API or direct URLs.

## Icon Requirements
- Format: PNG or JPG
- Size: 64x64px to 128x128px recommended
- Style: Food/restaurant themed icons

## Quick Start
For now, the application will use placeholder paths. Once you add the actual icon files with the correct names, they will automatically be used.
